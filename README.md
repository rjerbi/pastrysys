## Pastrysys :
Pastrysys is a Pastry Management System web application for managing a pastry shop. This system allows managing products, categories, sales, users, and suppliers.

## Features :
**User Management**
  - Admin and Employee roles.
  - Add, edit, delete users.
  - Profile editing and password change.

**Product Management**
  - Add, edit, delete products.
  - Manage categories and product images.
  - Track stock quantity and prices.

**Sales Management**
  - Record daily sales.
  - Update sales.
  - Track total revenue.

**Supplier Management**
  - Add, edit, delete suppliers.
  - Store supplier contact and company information.

**Reporting**
  - Daily sales overview.

## Database :
The project uses a MySQL database named `systeme_patesserie` with the following tables:

- `users` – store user information.
- `user_groups` – store roles (Admin, Employee).
- `categories` – product categories.
- `products` – store products and prices.
- `media` – product images.
- `sales` – sales records.
- `suppliers` – supplier information.

## Technologies Used :
- **Backend:** PHP
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** MySQL
- **Other:** JavaScript, jQuery
